How To Install LAMP Server In Ubuntu 12.04

May 8th, 2012 Miles Dyson
8,499 views
Leave a comment   

LAMP is one of the most common web hosting platforms on the Internet, so it’s a great environment to build and test your website.

LAMP stands for “Linux, Apache, MySQL, PHP”

The developers at Ubuntu have made it verry easy to install and configure the packages for LAMP with a single terminal command.

 

1. Open Terminal (ctrl+alt+t).

2. Type in this command.
sudo apt-get install lamp-server^

3. Type in password press enter.
